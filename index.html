Course Project • Individual Project

Tank War Game AI Based on Q-Learning

Featured
JavaScript HTML5 Canvas Q-Learning MDP Hyperparameter Tuning

Time

Course Project

Problem

Build a custom 1v1 tank battle web game and train an agent that can improve decision-making in combat rather than rely on fixed scripted behavior.

Data

Data came from repeated training episodes in the custom game environment, including reward signals, win/loss outcomes, moving average performance, epsilon decay, and behavior changes across 1,000 training matches per experiment group.

Approach

I implemented Q-learning from scratch with JavaScript and HTML5 Canvas, formulated the reinforcement learning task as an MDP with a discrete action space, applied reward shaping and epsilon-greedy exploration, and built a real-time dashboard for rewards, epsilon decay, hyperparameters, and performance charts. I also tuned five hyperparameter groups and addressed reward hacking and false convergence issues.

Outcome / Impact

The best configuration used Alpha = 0.18 and Gamma = 0.92. The report recorded a moving average win rate of 94%, with 854 wins out of 1000 matches, showing stable improvement after iterative tuning and debugging.

My Contribution

This was an individual project, so I designed and implemented the game logic, reinforcement learning setup, visualization dashboard, experiment tuning, and issue diagnosis independently.

Academic Project • Group Project

Lingnan CrownCred (EduReward)

Group Work
Next.js 13 TypeScript Chakra UI Ethers.js Wagmi React Query

Time

Academic Group Project

Problem

Design a blockchain-based academic incentive prototype that connects academic performance with a transparent reward mechanism for eligible students.

Data

The system uses off-chain grade records entered through the frontend and JSON storage, then connects eligibility results to on-chain reward claims and NFT minting actions.

Approach

The project used a hybrid off-chain and on-chain design. Smart contracts handled reward distribution, claim status, and event logging, while the frontend supported grade entry, profile management, reward claiming, redemption flows, and NFT minting. The report states that the frontend stack used Next.js 13, TypeScript, Chakra UI, Ethers.js, Wagmi, and React Query. Static analysis with Slither and functional testing were included, and the system was deployed on the Sepolia testnet.

Outcome / Impact

The result was a working prototype showing how academic records can be linked with ERC-20 reward claims and ERC-721 NFT minting through MetaMask in a structured incentive flow.

My Contribution

This was a group project. I participated as a group member and contributed to collaborative work around blockchain incentive design, implementation support, testing, and project analysis, without claiming sole ownership of the system.
