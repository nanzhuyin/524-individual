# 524 Individual Project – Personal Job-Seeking Website

This repository contains my personal job-seeking website for the CDS524 individual project. The website is built as a static site and deployed with GitHub Pages.

## Website Purpose

This website is designed to present my:

- job target and personal positioning
- background and skills
- featured academic projects
- resume / CV
- contact information

It serves as an online portfolio for internship applications in:

- AI Product
- Blockchain / Web3
- Data Analysis

## Included Sections

The website includes the following required sections:

1. Home / Headline  
2. About Me  
3. Skills  
4. Projects  
5. Resume / CV  
6. Contact  

## Featured Projects

### 1. Tank War Game AI Based on Q-Learning
An individual project that implements a 1v1 web-based tank battle game using Q-learning. The project includes reinforcement learning design, reward shaping, training visualization, and hyperparameter tuning.

### 2. Lingnan CrownCred (EduReward)
A group project that explores a blockchain-based academic incentive system using off-chain grades and on-chain ERC-20 / ERC-721 rewards, with frontend interaction and Sepolia testnet deployment.
